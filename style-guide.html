Portfolio Style Guide

Overall Formatting

Body uses Segoe UI, soft pink background (#ffeef6), 40px margins, and #333 text.

Site Header / Branding

Header title uses Playfair Display, centered, deep pink, with letter spacing.

Navigation Links

Navigation bar uses #ffd6e8 background, centered links, bold rose-colored text (#b3005e).

Content Section

Main content card is white with rounded corners, pink shadow, and 1150px max width.

Site Footer

Footer is fixed to the bottom-left, soft pink background, and bold rose link styling.
